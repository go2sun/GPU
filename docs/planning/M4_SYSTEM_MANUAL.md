# 🛠️ M4 视觉审计系统 (最终版) 使用手册

## 🚀 极速指令集
- `syncm4`: 全局归位与云端同步。
- `m4-quiz`: 启动 AI 维修知识测评。
- `m4-mats`: 自动化显存故障审计。
- `m4run [脚本名]`: 极速注入执行 sync_tools 脚本。

## 📁 目录架构规范
- `/scripts/`: 所有 Python/Shell 逻辑核心。
- `/data/logs/`: 存放 MATS/MODS 原始日志。
- `/docs/`: 存放审计报告与学习路径。

## ⚖️ 审计准则
- **100% 中文输出**：保持审计的可读性。
- **物理映射**：所有软件报错必须对应到硬件物理坐标。
